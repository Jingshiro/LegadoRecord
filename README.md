# Legado Record

Legado (阅读) 本地数据可视化分析。

[Live Demo](https://jingshiro.github.io/LegadoRecord/)

## 使用

1. 导出：在 APP 内导出备份（一般命名为backup[时间]-设备.zip），解压 `.zip` 包找到 `readRecord.json`。
2. 运行：打开 [在线版](https://jingshiro.github.io/LegadoRecord/) 或本地 `index.html`。
3. 导入：拖入文件即可。
4. AI：如需 AI 分析，请在设置中填入 Base URL 和 Key。
