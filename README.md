# Legado Record

Legado (阅读) 本地数据可视化分析。

[Live Demo](https://jingshiro.github.io/LegadoRecord/)

## 使用

1. 导出：用这个[阅读魔改版](https://github.com/Jingshiro/legado)，在我的-阅读记录-导出详细记录里获得json
2. 运行：打开 [在线版](https://jingshiro.github.io/LegadoRecord/) 或本地 `index.html`。（本地可能会有CORS问题）
3. 导入：拖入文件即可。
4. AI：如需 AI 分析，请在设置中填入 Base URL 和 Key。
